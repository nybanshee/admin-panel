# Project Requirements: Animated Admin Panel

## 1. Project Overview
Build an animated Admin Panel with a focus on UI/UX, featuring a sidebar for navigation and five primary views: Overview, Logs, Economy, Game Settings, and Graphs. The aesthetic should be "Anime Dark" with cool shapes, utilizing a Blue, Black, and Gray color palette.

## 2. Core Features
- **Sidebar Navigation**:
  - Located on the left side.
  - Animated transitions.
  - Links to: Overview, Logs, Economy, Game Settings, Graphs.
- **Overview Page**:
  - Dashboard summary with widgets.
- **Logs Page**:
  - Console/terminal emulator interface.
- **Economy Page**:
  - Management or visualization of game economy stats.
- **Game Settings Page**:
  - Global switches for systems (e.g., GunSystem).
  - Specific switches for individual items (e.g., each Gun).
- **Graphs Page**:
  - Visualization of "Progression".
  - Visualization of "Level to Unlocks".

## 3. Tech Stack
- **Frontend**: React, TypeScript, Vite.
- **Styling**: Tailwind CSS.
- **Animation**: Framer Motion.
- **Routing**: React Router DOM.
- **Icons**: Lucide React.
- **Charts**: Recharts.

## 4. User Interface
- **Theme**: "Anime Dark" aesthetic.
  - Colors: Deep Blacks, Cool Grays, Electric Blues.
  - Shapes: Angular, geometric, possibly "tech" or "sci-fi" inspired borders.
- **Responsiveness**: Desktop-first.
